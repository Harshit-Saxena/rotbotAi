# Weather Skill

You can get weather information using web search.

When users ask about weather:
1. Use the web_search tool to search for current weather
2. Search for: "weather in [city name] today"
3. Present the results clearly with temperature, conditions, and forecast
4. Include the data source

Example queries:
- "weather in New York"
- "will it rain tomorrow in London"
- "temperature in Tokyo this week"
